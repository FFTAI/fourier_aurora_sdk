# 初始状态参考

**初始状态**是 Aurora 的进入状态，其中没有控制器运行。

## 状态明细

状态名称 | 任务名称 | 手柄映射 | DDS 映射 | 频率
-----------|-----------|------------------|-------------|-------------
Default    | （无任务） | LB+RB            | 0           | -

可用于悬挂 | 可用于站立 | 自动保护切换
----------------------|------------------------|----------------
是                   | 否                     | 否

## 手柄控制

### 进入初始状态

初始化 *AuroraCore* 后，同时按下肩键 `LB` 和 `RB` 进入初始状态。

## 客户端控制

速度控制 | 站姿控制 | 关节控制 | 关节参数控制
-----------------|--------------------|---------------|-------------------
否               | 否                 | 否            | 否

### 进入初始状态

初始化 *AuroraCore* 后，使用 aurora 客户端的 `set_fsm_state` 函数进入初始状态。

```python
client = AuroraClient.get_instance(domain_id=123, robot_name="fouriern1")   # 初始化 aurora 客户端
time.sleep(1)

client.set_fsm_state(0)     # 切换到初始状态
```
